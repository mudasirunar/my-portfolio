# Mudasir's Portfolio Website

🌐 **Live Site:** [my-portfolio.unarmudasir.workers.dev](https://mudasir.tech)

A personal developer portfolio built with **React**, **TypeScript**, and **TanStack Router**. Showcases my projects, skills, and experience as a Junior Android Developer.

## Tech Stack

- **React 19** + **TypeScript**
- **TanStack Router** (file-based routing)
- **Tailwind CSS v4**
- **Framer Motion** (animations)
- **Vite** (build tool)
- **EmailJS** (contact form)

## Features

- Responsive design with smooth animations
- Project showcase with image gallery
- Downloadable PDF resume generator
- Working contact form


## Deployment

Configured for **Cloudflare Pages** via `wrangler.jsonc`.
